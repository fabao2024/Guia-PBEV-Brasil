Política de Privacidade

Última atualização: 13 de março de 2026

1. Quem somos

O Guia PBEV Brasil (fabao2024.github.io/Guia-PBEV-Brasil) é um catálogo interativo de veículos elétricos (BEVs) disponíveis no mercado brasileiro, com dados oficiais de autonomia PBEV/Inmetro. O serviço é operado por pessoa física, com contato disponível em guiapbevbrasil@gmail.com.

2. Dados coletados

Não coletamos dados pessoais identificáveis. Não há cadastro, login ou formulário de dados pessoais neste site.

Os únicos dados processados são:

  • Preferências locais (idioma, estado selecionado, favoritos, chave de API) — armazenadas exclusivamente no seu navegador via localStorage. Nunca enviadas a nossos servidores.
  • Dados de navegação anônimos — coletados pelo Plausible Analytics (ver seção 3).

3. Analytics — Plausible

Utilizamos o Plausible Analytics, uma ferramenta de análise de tráfego que:

  • Não utiliza cookies
  • Não rastreia usuários individualmente
  • Não compartilha dados com terceiros
  • É compatível com a LGPD sem necessidade de banner de consentimento

Os dados coletados são exclusivamente agregados: número de visitas, páginas mais acessadas e eventos de interação (ex: modelos visualizados, filtros aplicados). Nenhuma informação permite identificar um usuário específico.

4. Links de afiliados

Alguns botões do site (como "Cotar seguro EV" e "Ver na concessionária") são links de afiliado. Ao clicar, você é redirecionado ao site do parceiro, que tem sua própria política de privacidade. Podemos receber uma comissão caso você contrate um serviço após o clique.

Os links incluem parâmetros UTM (utm_source=guiapbev) para rastreamento de origem — esses parâmetros não coletam dados pessoais seus.

5. Chatbot de IA (Gemini)

O assistente de IA utiliza a API do Google Gemini. A chave de API que você fornece é armazenada apenas no seu navegador (localStorage) e nunca é transmitida aos nossos servidores.

As mensagens enviadas ao chatbot são processadas diretamente pela API do Google. Consulte a Política de Privacidade do Google para entender como os dados do Gemini são tratados.

6. Cookies

Este site não utiliza cookies. As preferências do usuário são salvas em localStorage, que fica restrito ao seu dispositivo e não é acessível por terceiros.

7. Seus direitos (LGPD)

Nos termos da Lei Geral de Proteção de Dados (Lei nº 13.709/2018), você tem o direito de:

  • Acessar, corrigir ou excluir seus dados — no caso deste site, basta limpar o localStorage do navegador (Configurações → Privacidade → Limpar dados do site)
  • Solicitar esclarecimentos sobre o tratamento de dados pelo e-mail abaixo
